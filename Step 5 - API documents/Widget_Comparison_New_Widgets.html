Modern API Codebase

✨ NEW — Phase 2 Widget
Widget Identity
Keykpi-cards

Module FieldCrossModule
Data API
GET /api/dashboard/kpi-cards
Returns FinancialKpiCardsDto {ResolvedPeriod,PriorYearPeriod,Income,Expenses,NetIncome,CashPosition}.
Period Resolution Logic
Uses X-Year-ID header or most recent open fiscal year
Uses X-Period-ID header or most recent non-Audit period
Prior year: fiscal year immediately before current, matching period ordinal number
Cash Position
SUM(all company bank accounts ending balance)
EndingBalance = LastReconcile.EndingBalance + SUM([BR_Item].ReconcileID=null)
Runway: trailing 3 periods avg expenses → currentCash/avgMonthly
Unreconciled warning: accounts >30 days since last reconcile

EF Core Pattern
Multi-step async query pipeline (9 steps):
1. Resolve current year+period 2. Resolve prior year+period 3. Scope income/expense account IDs
4. Current period actuals (income+expense) 5. Prior year period actuals
6. Current cash position (all bank accounts) 7. Prior year cash (last reconcile in prior year)
8. Cash runway calc (trailing 3 periods expenses) 9. Unreconciled account warnings
