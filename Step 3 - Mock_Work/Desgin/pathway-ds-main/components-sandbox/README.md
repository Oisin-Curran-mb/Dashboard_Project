# Components Sandbox

Experimental prototypes for the Pathway Design System. These are sketches, not production. Every prototype here is a self-contained HTML file that can be opened directly in a browser. Do not link to these files from docs, specs, or Storybook.

When a prototype matures into a real component, it is designed in Figma, then shipped via `/pathway:component-pipeline`, and then the corresponding sandbox file is moved to an `archived/` subfolder (manual step).
