# Pathway Design System

The component library for **Ministry Brands Amplify** — a church-management product.

Use the left sidebar to explore:

- **Tokens / Primitives** — raw colour palettes, type scales, units
- **Tokens / Semantics** — light-mode + dark-mode tokens, layout values, typography sets
- **Library** — interactive component playgrounds: SideNav, TopNav, Checkbox, OrgSwitcher, Spinner

Each component has a docs page with anatomy, design tokens, state matrix, accessibility notes, and a live Playground with controls.

---

### Building with an AI agent?

AI agent instructions and the machine-readable component contract live in the repo, not here.
Point your agent at:

- `AGENTS.md` — top-level agent guide (industry-standard filename, auto-discovered by Cursor, Codex, Aider, etc.)
- `llms.txt` / `llms-full.txt` — Jeremy Howard standard for AI-readable site/repo manifests
- `components/manifest.json` — machine-readable component registry
- `components/<name>/agent-brief.md` — per-component concise brief

Repo: [github.com/helloimjolopez-collab/pathway-ds](https://github.com/helloimjolopez-collab/pathway-ds)
