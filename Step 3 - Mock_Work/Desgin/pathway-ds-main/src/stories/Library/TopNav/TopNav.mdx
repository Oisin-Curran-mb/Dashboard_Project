# TopNav.Global

The fixed horizontal bar that sits at the top of every page in Ministry Brands Amplify. It persists across all modules and all screen sizes. Three responsibilities: show which module the user is in (ModuleSwitcher), show which organization the user is operating as (OrgSwitcher), and give quick access to search, notifications, and the user profile.

**Always on the brand-blue surface.** All interactive controls use dark-mode tokens regardless of the app's active color scheme.

**Spec:** `components/top-nav/top-nav-spec.md` · **Figma:** [Open node 40007067-6508](https://www.figma.com/design/3sw45aVcngFAmpbP6cfrXP/?node-id=40007067-6508) · **[Live demo](https://helloimjolopez-collab.github.io/pathway-ds/components/top-nav/top-nav.html)**

## Accessibility

- All interactive controls have 48×48px touch targets (WCAG 2.5.5).
- ModuleSwitcher and OrgSwitcher use the Disclosure (Show/Hide) ARIA pattern with `aria-expanded` on every trigger.
- On mobile, the OrgSwitcher button carries `aria-label` with the full unabbreviated org name even though the visual label is abbreviated.
- Escape closes any open panel and returns focus to its trigger.
- Focus ring: `2px solid rgba(160,181,230,0.7)` at `outline-offset: 2px`.
